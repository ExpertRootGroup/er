Рассмотрим один модуль.
Прилетает нейтрон с энергией 500 МэВ.
Точка входа равномерно размазана по сечению одного из файберов размером 6х6 мм.
Range cut = 0.7 мм.
Интересны энерговыделение и световыход, отсуммированные в каждом файбере.
Записывать обе величины надо в случае, если энерговыделение в файбере превосходит 100 кэВ.
Суммирование величин по пойнтам внутри файбера по идее не нуждается в нижнем пороге.
